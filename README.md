# multilocale

[https://www.multilocale.com](multilocale) CLI to translate strings in native and web apps.

## Download

Run `$ multilocale download` to download all the translation files locally.

## Import

Run `$ multilocale import` to import existing translation files into [https://app.multilocale.com](multilocale.com).

## Add

Run `$ multilocale add 'SOME_KEY' 'SOME_VALUE'` to add a string.
If the key and value are the same you can just run `$ multilocale add 'SOME_KEY'`
